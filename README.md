# CosmosExplorer 

A website to explore space images using NASA's public API.

## About the Project

CosmosExplorer shows Astronomy Picture of the Day (APOD) 
and Mars Rover photos fetched from NASA's free API. 
Users can search by keyword, filter by date range, 
and sort the results.

## Features

- Browse NASA Astronomy Pictures of the Day
- View Mars Rover photos from Curiosity
- Search by keyword
- Filter by date range
- Sort by newest or oldest

## Techstack Used

- HTML
- CSS
- JavaScript
- NASA Public API

## API Used

NASA API - https://api.nasa.gov/
